# AutoGestorFacturas

Extractor de datos de facturas usando Google Gemini AI.

## Instalación

1.  Clonar repositorio.
2.  Instalar dependencias:
    ```bash
    pip install -r requirements.txt
    ```
3.  Configurar `.env`:
    Copia `.env.example` a `.env` y añade tu `GEMINI_API_KEY`.

## Uso

```bash
python main.py "ruta_al_archivo.pdf"
```
